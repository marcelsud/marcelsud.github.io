Tech Translations

29 January 2016 on open source

An open source attempt to disseminate high-quality content in many languages

I've been thinking a lot about how I could help the software developers from my country (Brazil) to learn from the plenty of content published by the software development community every day. So I decided to start an open source project called "Tech Translations".

The project's idea is to provide high-quality content related to software development to those developers that do not speak English.

In the beginning, my intention was to translate only from English to Portuguese (pt_BR) but then I thought that maybe it would be a good idea to translate it to other languages too (with the community's help).

I am certain that if English is not your native language it was a little bit difficult to find high-quality content in the beginning of your career (or maybe you are facing that challenge now).

Usually, the content you find out there is outdated and obsolete. It is obvious that there is quality content written in your native language, but certainly it can not be compared with the content you will find in English.

The name may change in the future, but the idea will remain...

If you'd like to find out more about Tech Translations, check out the Github repository: https://github.com/marcelsud/techtranslations
